组件数据流

ImageUploader（封面图 - 单图）
Props: value: string（OSS URL 或空）+ onChange(url: string)
内部状态：uploading | preview | error
流程：选文件 → 压缩 → 获取 STS → OSS 直传 → 回调 onChange(ossUrl)

ImagesUploader（详情图 - 多图）
Props: value: {url, sortOrder}[] + onChange(images)
内部状态：上传队列、排序列表
流程：批量选择 → 逐个压缩上传 → 加入列表 → 拖拽更新排序
