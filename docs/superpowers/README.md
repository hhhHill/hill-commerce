# Superpowers Docs

`docs/superpowers/` 是本仓库的 AI 执行辅助文档根目录。

它长期保留并负责承载：

- `prompts/`
- `workflows/`
- `commands/`
- `checklists/`

注意：

- `docs/superpowers/specs/` 与 `docs/superpowers/plans/` 仅保留历史参考价值
- canonical product / architecture / implementation truth 已迁移到 `specs/*`
- 新的执行辅助文档应优先写入本目录，而不是重新写入 `docs/superpowers/specs/*` 或 `docs/superpowers/plans/*`
