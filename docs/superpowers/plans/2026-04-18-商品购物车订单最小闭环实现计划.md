# 商品、购物车与订单最小闭环实现计划

> **面向代理执行者：** 实施本计划时，必须按任务逐项执行，步骤使用复选框（`- [ ]`）跟踪状态。

**目标：** 基于现有 `hill-commerce` 模块化骨架，实现“商品查询 -> Redis 购物车 -> 创建订单”的最小真实下单闭环。

**架构方式：** `product` 负责商品持久化与查询，`cart` 以 Redis 作为主存维护购物车，`order` 基于勾选购物车项生成 MySQL 订单快照。控制器统一位于 `apps/backend-api`，模块之间仅通过对外 `api` 协作。

**技术栈：** Java 17、Maven、Spring Boot、Spring Web、Spring Data JPA、Spring Data Redis、MySQL、Redis、Flyway、JUnit 5
